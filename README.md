## About

This game is inspired by the MTC recruitment poster aesthetics:
- **Color scheme** matches the official MTC recruitment poster
- **Food items** feature MTC logos (safe to collect!)
- **Obstacles** are rival club logos (avoid these!)

## Gameplay

- Collect MTC logos to grow your snake and increase your score
- Avoid rival club obstacles - hitting them ends the game
- Obstacles randomize position each time you collect food
- Difficulty increases as your score grows (more obstacles spawn)

## Controls

- **Arrow Keys** or **WASD** - Move the snake
- **Space** - Pause/Resume the game

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open http://localhost:8080 in your browser.
